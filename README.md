### Level-1: 基础外设驱动

#### 1_UART - 串口通信
- 以 **1Hz** 频率定时发送 `"Hello World\r\n"`
- 接收用户输入，检测到回车时响应特定字符串
- 波特率 115200，8N1 配置

#### 1_I2S - 数字音频合成（**重点完成**）

本项目采用**模块化架构设计**，将驱动层与应用层分离：

```
main/
├── audio_generator.h/c     # 音频波形生成库
├── i2s_driver.h/c          # I2S驱动封装库
├── button_driver.h/c       # 按键驱动库（消抖+长按检测）
├── task_manager.h/c        # 实验任务管理器
└── main.c                  # 主程序入口
```

**三个实验任务全部实现**：

| 任务 | 功能描述 | 现象 |
|:----:|:---------|:-----|
| 任务1 | 右声道输出 **500Hz锯齿波** | 听到锯齿波音色 |
| 任务2 | 左1001Hz + 右999Hz正弦波，L+R模式 | 听到 **2Hz拍频**（音量周期性起伏）|
| 任务3 | 左1kHz + 右4kHz正弦波，BOOT键切换声道 | 支持左/右/双声道切换，长按3秒返回任务1 |

**核心算法**：
- 锯齿波生成：`sawtooth = 2.0f * phase - 1.0f`
- 正弦波生成：`sample = AMPLITUDE * sinf(2π * phase)`
- 拍频原理：两个相近频率叠加产生 `(f₁-f₂)` 的包络频率

**技术收获**：
- 理解 I2S 协议时序（BCLK、WS、DATA）
- 掌握 DMA 音频数据传输
- 实现状态机按键检测（短按/长按识别）
- 模块化编程思想（高内聚、低耦合）

## 📚 技术栈总结

### 硬件平台
- **ESP32-S3** (Dual-core Xtensa LX7, 240MHz)
- **MAX98357A** I2S功放模块

### 软件框架
- **ESP-IDF v5.0+** (Espressif IoT Development Framework)

| 类别 | 掌握内容 |
|:-----|:---------|

| 通信协议 | UART、I2S、SPI、WiFi |
| 文件系统 | FATFS (SD卡) |
| 网络协议 | TCP/IP、DHCP、NAT |
| 音频处理 | 波形合成、PCM、MP3解码 |
| 工程工具 | Git、CMake、menuconfig |

## 🚀 如何编译运行

### 环境准备

```bash
# 1. 安装ESP-IDF（v5.0或更高版本）
git clone -b v5.0 --recursive https://github.com/espressif/esp-idf.git
cd esp-idf
./install.sh esp32s3
. ./export.sh

# 2. 克隆本项目
cd ~
git clone <repository-url> 2026Spring-4-Dian
cd 2026Spring-4-Dian
```

### 编译任意Level项目

```bash
# 进入目标项目目录
cd Level-1/1_I2S

# 设置目标芯片
idf.py set-target esp32s3

# 编译
idf.py build

# 烧录（替换为实际串口）
idf.py -p /dev/ttyUSB0 flash

# 查看日志
idf.py monitor

# 或一步完成
idf.py build flash monitor
```
