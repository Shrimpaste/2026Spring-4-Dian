# ESP32-S3 MP3 Player with SD Card

基于 ESP32-S3 的 MP3 播放器，使用 **SD 卡存储**、I2S 音频输出和串口上位机控制。

| NVS 配置 | ✅ 正常 | 首次启动命名、配置保存正常 |
| I2S 音频 | ✅ 已编译 | 待硬件测试 |

### 已知问题

1. **SD 卡无法挂载** - 初始化时返回错误，可能原因：
   - SPI 引脚配置与实际硬件不匹配
   - SD 卡模块供电不足
   - 接线松动或接触不良
   - SD 卡格式非 FAT32

2. **MP3 解码未验证** - minimp3 已集成但未实际测试：
   - 需等待 SD 卡修复后才能读取 MP3 文件
   - 解码后的 PCM 输出到 I2S 待验证

## 功能特性

### Level 2-1-1: 首次上电取名
- 首次启动提示用户为 MP3 播放器设置名称
- 名称保存到 NVS Flash，之后上电不再提示
- 使用命令 `name <名称>` 设置

### Level 2-1-2: SD 卡歌曲管理
- 自动扫描 SD 卡中的 MP3 文件
- 显示歌曲数目和名称列表
- 歌曲信息存储在内存中

### Level 2-2: I2S 音频播放
- 使用 MAX98357A I2S 功放模块
- 支持 44.1kHz 采样率立体声输出
- 集成 **minimp3** 官方 MP3 解码库
- 软件音量控制
- 自动检测并适配不同采样率的 MP3 文件

### Level 2-3-1: 播放控制
- 串口上位机命令控制
- 支持：播放、暂停、继续、停止、切歌
- 音量调节 (0-100%)

### Level 2-3-2: 断点记忆
- 切歌时保存播放位置
- 重新切回时从断开处继续播放
- 播放状态持久化到 NVS

## 硬件要求

| 模块 | 型号 | 说明 |
|------|------|------|
| 开发板 | **ESP32-S3 N8R8** | 8MB Flash, 8MB PSRAM 版本 |
| SD卡模块 | SPI接口 | 用于存储 MP3 文件 |
| I2S功放 | MAX98357A | I2S 数字功放模块 |
| 扬声器 | 4Ω 3W | 或类似规格 |
| USB线 | - | 用于供电和调试 |

### 详细接线表

#### SD 卡模块接线 (SPI)

| SD卡模块 | ESP32-S3 | 功能说明 |
|----------|----------|----------|
| MOSI     | **GPIO4** | SPI 数据输入 |
| MISO     | **GPIO5** | SPI 数据输出 |
| SCK/CLK  | **GPIO2** | SPI 时钟 |
| CS       | **GPIO8** | SPI 片选 |
| VCC      | 3.3V/5V  | 电源输入 |
| GND      | GND      | 地线 |

#### MAX98357A I2S 功放模块接线

| MAX98357A | ESP32-S3 | 功能说明 |
|-----------|----------|----------|
| BCLK      | **GPIO10** | I2S 位时钟 |
| LRCK (LRC)| **GPIO11** | I2S 左右声道时钟 |
| DIN       | GPIO6    | I2S 数据输入 |
| SD_MODE   | GPIO7    | 功放使能 (高电平有效) |
| GND       | GND      | 地线 |
| VIN       | 3.3V/5V  | 电源输入 |

#### 扬声器接线

| 扬声器 | MAX98357A |
|--------|-----------|
| + (正极)| OUT+      |
| - (负极)| OUT-      |

## 软件依赖

- ESP-IDF v5.0 或更高版本
- 支持的组件:
  - `fatfs` - FAT 文件系统 (用于 SD 卡)
  - `sdmmc` - SD/MMC 驱动
  - `nvs_flash` - 非易失性存储
  - `driver` - GPIO/I2S/SPI 驱动
  - `freertos` - 实时操作系统

## SD 卡要求

| 参数 | 要求 | 说明 |
|------|------|------|
| 容量 | 最大 32GB | FAT32 文件系统 |
| 文件格式 | FAT32 | 使用 FAT 文件系统 |
| 文件类型 | MP3 | 标准 MP3 文件 |
| 建议码率 | 128-192 kbps | 平衡音质和文件大小 |

## 串口命令

连接串口终端 (波特率 115200)，使用以下命令控制播放器:

| 命令 | 参数 | 说明 |
|------|------|------|
| `play` | [n] | 播放 (n=歌曲编号) |
| `pause` | - | 暂停播放 |
| `resume` | - | 继续播放 |
| `stop` | - | 停止播放 |
| `next` | - | 下一首 |
| `prev` | - | 上一首 |
| `list` | - | 显示歌曲列表 |
| `vol` | 0-100 | 设置音量 |
| `status` | - | 显示播放器状态 |
| `storage` | - | 显示 SD 卡存储信息 |
| `name` | <名称> | 设置播放器名称 |
| `help` | - | 显示帮助 |

    SD Card Configuration  --->

    I2S Configuration  --->
        I2S BCLK GPIO (10)

        Audio Sample Rate (44100)

## 注意事项

1. **引脚冲突**: GPIO19 和 GPIO20 保留给 USB 使用，请勿用于其他功能
2. **SD 卡格式**: 请确保 SD 卡格式化为 FAT32 格式
3. **电源**: 确保电源稳定，建议使用独立 5V 电源为 MAX98357A 供电
4. **文件格式**: 支持标准 MP3 文件，建议 128-192kbps

| SD SCK | GPIO2 | SPI 时钟 |

| I2S BCLK | GPIO10 | 位时钟 |
| I2S LRCK | GPIO11 | 声道时钟 |
| I2S DATA | GPIO6 | 数据输出 |

## 播放格式说明

本版本使用 **minimp3** 官方库进行 MP3 解码：

1. **支持格式**: 标准 MP3 文件 (MPEG-1 Layer 3)
2. **采样率**: 自动检测并适配 (支持 32000/44100/48000 Hz)
3. **声道**: 支持单声道和立体声 (自动转换为立体声输出)
4. **解码库**: minimp3 (轻量级、高性能的纯 C MP3 解码器)
5. **输出**: 16-bit PCM，通过 I2S 输出到 MAX98357A

## 故障排除

### SD 卡无法挂载
- 检查 SD 卡格式是否为 FAT32
- 检查 SPI 接线是否正确（MOSI/MISO/SCK/CS）
- 确认 SD 卡模块供电正常
- 尝试更换 SD 卡

### 无声音输出
- 检查 MAX98357A 的 SD_MODE 引脚是否为高电平
- 确认扬声器连接正确
- 检查音量设置（默认 70%）
- 验证音频文件格式（需要 16位 PCM，44.1kHz，立体声）

### 串口无响应
- 检查波特率是否为 115200
- 确认使用正确的 USB 转串口连接
- 尝试复位开发板
